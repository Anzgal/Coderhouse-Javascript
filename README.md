# Coderhouse Javascript
Este es el proyecto final para el curso de Javascript en Coderhouse.
El proyecto consiste en un quiz
Conceptos aplicados:
- Ciclos
- Funciones
- Arrays
- Funciones de orden superior
- Manipulación de DOM
- Captura de eventos
- Local Storage
- JSON
- Syntactic Sugar
- Librerías (Sweet Alert 2)
- Async
- Fetch
